# Welcome to F2M TV Player

Your favourite IPTV player for **Android phones** and **Android TV**.

---

## Features

- Live TV channels with group tabs
- Movies section
- Auto-update detection from GitHub
- Built for both phones and TVs (D-pad remote support)

## How to Use

- Open the app and wait for channels to load
- Use the **TV** tab for live channels
- Use the **Movies** tab for movies
- Use the **About** tab for app info
